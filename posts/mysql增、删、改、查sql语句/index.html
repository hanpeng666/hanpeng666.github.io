mysql增、删、改、查sql语句

mysql表内容操作 增、删、改、查

一、增

insert into 表 (列名，列名…) values (值，值,…)[,(值，值，…),…]
insert into 表 set 列名1=””,列名2=””,…
insert into 表 (列名，列名…) select (列名，列名…) from 表

例如：
insert into tutors set tname=’tom’,gender=’f’,age=30;
insert into tutors (tname,gender,age) select name,gender,age from student where age >20;

二、删

delete from 表 #删除表里所有数据
delete from 表 where name=”shaochao” #删除name=”shaochao”那一行数据

三、改

update 表 set name=”zhansan” where id>8;

四、查

查这块的条件太多了，我简单列举几个常用的

一、简单查询

a、条件判断where
WHERE子句：
1.布尔关系表达式 =、>、>=、<=、<
2.逻辑关系：AND OR NOT
select from 表 where id > 1 and name !=’hanshao’;
select from 表 where time between ‘2017-08-04 00:00:00’ and ‘2017-08-08 10:30:00’;
select from 表 where id in (11,22,33);
select from 表 where id in (select id from 表2);

b、通配符like
%: 任意长度任意字符
_：任意单个字符
select name from student where name like ‘y%’; #查询名字以y开头的
select name from student where name like ‘y__’; #查询名字以y开头长度为3的
select name from student where name like ‘%ing%’; #包含ing的名字
rlike 正则匹配
select name from student where name rlike ‘^[mny].*$’; #正则匹配以m 或 n 或y开头的名字

c、限制limit
select from 表 limit 5; #前5行
select from 表 limit 2,3; #从第2行开始的3行

d、排序asc,desc
select from 表 order by 列 asc; #默认 升序 从小到大排列
select from 表 order by 列 desc; #降序 从大到小排列
select * from 表 order by 列1 desc,列2 asc #根据列1 从大到小排列，如果相同则按列2从小到大排列

e、聚合：SUM(), MIN(), MAX(), AVG(), COUNT()
select avg(age) from students; 所有同学的平均年龄
select max(age) from students; 年龄最大
select min(age) from students; 年龄最小
select sum(age) from students; 年龄之和
select count(age) from students; 一共有多少人

f、分组group by 主要目的进行聚合计算）
select num from 表 group by num
select avg(age)from student group by gender(性别）; 男同学跟女同学的平均年龄
特别的：group by 必须在where之后，order by之前

g、HAVING qualification(过滤条件）
select count(cid1) as persons,cid1 from student group by cid1 having person > 2;(having可以直接引用别名做比较，having只能在group by里面使用，相

当于where，用于将group by分组结果再进行过滤)

h、查询结果不显示重复记录
SELECT DISTINCT 字段名 FROM 表名 WHERE 查询条件

二、多表查询

连接：
内连接：将两个表中存在连结关系的字段符合连接条件的记录形成记录集
select a.name,b.name from a inner join b on a.id=b.id;
select a.name,b.name from a join b on a.id=b.id;
结果是一样的（内连接的inner关键字可省略）
外连接：
左外连接：… LEFT JOIN … ON … (以左表为基准，右表没有的左表有的显示null，保存左表的全部记录)
sselect a.id,b.age from a left join b on a.id=b.id;
右外连接: … RIGHT JOIN … ON …
自连接：(回到自身找答案的)
select c.name as teacher,c.name as student from students as s,students as c where s.sid=c.tid;(同一个表查询）

联合查询：UNION

（select name,age from student）union (select name,age from tutors); 两个表查询的结果都显示在一起

三、子查询

select name from student where age > (select avg(age)from student);
IN(): 使用子查询；
select name from student where age in (select age from tutors);
在FROM中使用子查询；
select name,age from (select name,age from student)as t where t.age >= 20; (在查询出来的结果再进行比较查询）
